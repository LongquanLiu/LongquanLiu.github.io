9.变态跳台阶

一只青蛙一次可以跳上1级台阶，也可以跳上2级……它也可以跳上n级。求该青蛙跳上一个n级的台阶总共有多少种跳法。
知识点：动归

1) n级台阶时，会有n种跳的方式，1阶、2阶…n阶，得出结论：
f(n) = f(0) + f(1) + f(2) + f(3) + … + f(n-1)
f(n-1) = f(0) + f(1) + f(2) + f(3) + … + f(n-2)
可以得出：
f(n) = 2*f(n-1)

11.二进制中1的个数

reference,负数的二进制表示方法

原码：一个正数，按照绝对值大小转换成的二进制数；一个负数按照绝对值大小转换成的二进制数，然后最高位补1。
负数的反码：为对该数的原码除符号位外各位取反。
负数的补码为对该数的原码除符号位外各位取反，然后在最后一位加1（反码最后一位+1）
正数的反码和补码都与原码相同
负数在计算机中以原码的补码形式表达

c++/java:
>> 是带符号右移，若左操作数是正数，则高位补“0”，若左操作数是负数，则高位补“1”.
<< 将左操作数向左边移动，并且在低位补0.

12.数值的整数次方

快速幂就是把指数进行一次log(N)级别的变换。
假设求3^11, 11 = 2^3+2^1+2^0
那么我只需要算3^1和3^2还有3^8这样复杂度就降下来了。算3^1需要一次记为a,把 a平方 就是3^2记为b，把 b平方 就是3^4记为c,再平方就是3^8记为d，这样把a b d乘以之后就是结果了。这样速度就快起来了~

  • 头插法2，不多加临时头结点：，初始节点当做临时表头节点，将链表分为已反转部分和待反转部分，用两个指针分别指向已反转部分链表头tempHead、链表尾tempTail，未反转部分链表头tempTail.next;遍历tempTail.next,将每个节点插入到tempHead之后。最后将tempHead.next节点作为头结点，将原本tempHead插入到链表尾部！
  • 根据栈后进先出的性质，将链表元素全部压入栈，逐个压出组建新链表。

30.子数组的最大和

  • 思路：maxSum, curSum初始为第一个元素。
    遍历数组，curSum +=当前元素， 若大于curSum更新maxSum，否则继续。若curSum小于0重置为0。

31. 整数中1出现的次数

  • 思路：设定整数点（如1、10、100等等）作为位置点i（对应n的各位、十位、百位等等），分别对每个数位上有多少包含1的点进行分析,根据设定的整数位置，对n进行分割，分为两部分，高位partA=n/i，低位partB=n%i，该位对应的数为partA%10。
    • 当i表示百位，且百位对应的数>=2，如n=31456,i=100，则a=314,b=56，此时百位为1的次数有a/10+1=32（最高两位0~31）次是包含100个连续点，即共有(a/10+1)* 100个点的百位为1
    • 当i表示百位，且百位对应的数为1，如n=31156,i=100，则a=311,b=56，此时百位对应的就是1，则共有a/10=31(最高两位0-30)次是包含100个连续点，当最高两位为31（即a=311），本次只对应局部点00~56，共b+1次，即共有(a/10*100)+(b+1)，这些点百位对应为1
    • 当i表示百位，且百位对应的数为0,如n=31056,i=100，则a=310,b=56，此时百位为1的次数有a/10=31（最高两位0~30）次是包含100个连续点，即共有(a/10)* 100个点的百位为1

综合以上三种情况，当i位对应0或>=2时，有((a+8)/10) i；当百位为1时，有(a/10i)+(b+1)
ps:(a+8)/10，是因为当百位为0，则a/10==(a+8)/10，当百位>=2，补8会产生进位位，效果等同于(a/10+1)

50.数组中的重复数字

在一个长度为n的数组里的所有数字都在0到n-1的范围内。 数组中某些数字是重复的，但不知道有几个数字是重复的。也不知道每个数字重复几次。请找出数组中任意一个重复的数字。 例如，如果输入长度为7的数组{2,3,1,0,2,5,3}，那么对应的输出是第一个重复的数字2。

  • 思路1：先排序，之后
  1. 若只是判断是否重复，可从数组尾部向前遍历，判断是否值为定值。
  2. 判断重复，且找出一个元素，从数组前部开始遍历，若后一元素与现元素相等，记录（结束），否则后移，
  3. 判断重复，且找出所有对应元素，从数组前部开始遍历，若后一元素与现元素相等，记录（加入数组判断不重复），后移，
    特殊：输入数组为空，
    时间复杂度O(nlgn)

63. 数据流中的中位数

如何得到一个数据流中的中位数？如果从数据流中读出奇数个数值，那么中位数就是所有数值排序之后位于中间的数值。如果从数据流中读出偶数个数值，那么中位数就是所有数值排序之后中间两个数的平均值。我们使用Insert()方法读取数据流，使用GetMedian()方法获取当前读取数据的中位数。

标签：堆，数学

考虑用不同的数据结构保存，分析插入和返回中位数的时间复杂度。

  1. 无序数组保存，插入O(1)，返回中位数最优为O(n)，eg:
  2. 有序数组保存，插入可能O(n)，返回中位数为O(1)
  3. 排序的链表，插入O(n)，找定义两中位数指针O(1)
  4. 二叉搜索树，插入和查找平均O(lgn)，最坏O(n)
  5. AVL树，插入O(lgn)，O(1)找中位数

66. 机器人的运动范围

地上有一个m行和n列的方格。一个机器人从坐标0,0的格子开始移动，每一次只能向左，右，上，下四个方向移动一格，但是不能进入行坐标和列坐标的数位之和大于k的格子。 例如，当k为18时，机器人能够进入方格（35,37），因为3+5+3+7 = 18。但是，它不能进入方格（35,38），因为3+5+3+8 = 19。请问该机器人能够达到多少个格子？

标签：回溯法

  1. 从坐标（0,0）开始移动，当进入坐标为（i，j）的格子时，检查坐标的数位和判定是否能进入。若能进入则判断是否能进入4个相邻格子。
  2. 由于回溯递归特性，路径看做一栈，当在矩阵中走到第n个格子位置后，在第n个格子周围都没找到可进入相邻格子，则路径回溯到第n-1个格子，重新定位第n个格子；
  3. 由于路径不能重复，还需定义和矩阵大小一样的bool标识矩阵。
